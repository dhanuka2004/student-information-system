
# Student Information System

A Java-based console application to manage student records.  
This project demonstrates core concepts of Object-Oriented Programming (OOP), CRUD operations, and GitHub-based project management.

---

## 🚀 Features
- Add new students with details (ID, Name, Age, Course)
- View all student records
- Search students by ID
- Update student information
- Delete student records
- Simple console interface (future expansion: database + GUI)

---

## 🛠️ Technologies Used
- **Java** (Core language)
- **Collections (ArrayList)** for storing student data
- **Git & GitHub** for version control and collaboration
- *(Planned)* JDBC for database integration
- *(Planned)* JavaFX for GUI
